# FlowMark v0.1.4 Release Notes

Date: 2026-02-09

## Summary

FlowMark v0.1.4 adds non-normative guidance for references, evidence, and provenance. Validation rules remain unchanged.

## Highlights

- Recommended input references via `inputs`/`refs`
- Optional `evidence` fields for checklist items
- Provenance is recorded in aiwf sessions, not in FlowMark documents

## Compatibility

- v0.1.3 documents remain valid
- v0.1.4 documents remain structurally identical
