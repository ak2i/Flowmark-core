# FlowMark v0.1.2 Release Notes

Date: 2026-02-09

## Summary

FlowMark v0.1.2 adds guidance for running FlowMark workflows with aiwf (AI Workflow Framework). Validation rules are unchanged from v0.1.1.

## Highlights

- aiwf is introduced as the recommended execution environment
- Authoring pipeline steps are documented (contract draft/review/generate/validate/fix)
- Validator behavior unchanged from v0.1.1

## Compatibility

- v0.1.1 documents remain valid
- v0.1.2 documents remain structurally identical
