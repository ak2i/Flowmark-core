# FlowMark v0.1.5 Release Notes

Date: 2026-02-09

## Summary

FlowMark v0.1.5 adds a `describe` subcommand and bundles AI authoring guidance documents for stable, version-controlled prompts.

## Highlights

- `flowmark describe ai` outputs the prompt-friendly AI authoring guide
- `flowmark describe ai-source` outputs the canonical IdeaMark source
- CLI uses fixed guide paths under `docs/guides/`

## Compatibility

- No changes to FlowMark document structure
- Validators remain unchanged
