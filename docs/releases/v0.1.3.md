# FlowMark v0.1.3 Release Notes

Date: 2026-02-09

## Summary

FlowMark v0.1.3 clarifies spec versioning and the separation of aiwf as an external Runner. No structural or validation changes are introduced.

## Highlights

- Spec stack is now explicitly required in AI workflows
- aiwf is treated as an external Runner / Workflow Engine
- Validation rules unchanged from v0.1.2

## Compatibility

- v0.1.2 documents remain valid
- v0.1.3 documents remain structurally identical
